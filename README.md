# btc-eth-address-test

How to run:
```
cd btc-eth-address-test
python3 -m venv .
```
On Windows, run:
```
.\Scripts\activate.bat
```
On Unix or MacOS, run:
```
source ./bin/activate
```
Then: 
```
(btc-eth-address-test) $ python -m pip install -r requirements.txt
```
